3.배포 시 특이사항 : MySQL, Flask 서버를 먼저 올리는 게 좋을 것 같습니다.


4. EC2 계정에 들어갈 정보는 user : root / pass : a710soezmtc  입니다.


5. 파이어 베이스를 사용한 소셜 로그인(구글)은 구글 이메일로 들어가서 전화번호 인증만 해주시면 됩니다. 


6. DB 파일 : exec/ddl.sql 파일


7. exec/mtc_Test_case 파일로 확인이 가능합니다. 아래는 간단한 설명입니다.

7-1 : 구글 로그인을 추천드리며, 

7-2 : 만약 로그인 후 네브 외에 흰 배경만 보인다면 '인기' 혹은 '최신' 등의 버튼을 눌러주세요.

7-3 : 중앙에 사진이 나온다면 우측 하단에 있는 버튼을 통해 상 - 평 - 하 평가가 가능합니다.

7-4 : 사진을 아래로 내리면 해당 사진에 대한 댓글을 작성하거나 볼 수 있고

7-5 : 오른쪽으로 넘기면 다음 사진을 확인할 수 있습니다.

7-6 : 중앙 상단에 있는 메인 / 인기 / 최신 을 통해서 각기 다른 게시물 리스트를 확인할 수 있습니다.

7-7 : 상단 네브바 우측에 있는 프로필을 누르시면 마이 페이지 확인이 가능합니다.

7-8 : 마이 페이지를 들어간 상태에서 우측 상단에 설정 버튼을 누르면 세팅 페이지로 넘어갑니다.

7-9 : 게시물 혹은 댓글에 있는 프로필 사진을 누르면 다른 유저의 마이 페이지에 들어갈 수 있습니다.

7-10 : 다른 유저의 마이 페이지에서 해당 유저를 팔로우할 수 있습니다.

7-11 : 자신의 마이 페이지에서는 자신이 최근 작성한 게시물, 최근 본 게시물, 최근 평가한 게시물을 볼 수 있습니다.

7-12 : 다른 유저의 마이 페이지는 작성한 글만 확인이 가능합니다.

7-13 : 네브바 우측에는 알림 버튼이 있고, 다른 유저로부터 온 알림을 확인할 수 있는 페이지로 넘어갑니다.

7-14 : 알림 버튼 옆에는 게시물 작성 버튼이 있으며, 거기서 제목, 사진, 해시태그, 공개설정 등을 선택해서 게시물을 등록할 수 있습니다.

7-15 : 작성한 게시물은 보통 '최신'에서 확인이 가능합니다.

7-16 : 네브바 좌측에 있는 로고를 누르면 메인으로 돌아옵니다.

7-17 : 마이 페이지 -> 세팅 페이지 -> My Page 로 들어가면 자신의 정보를 수정할 수 있습니다.

7-18 마이 페이지 -> 세팅 페이지 -> 팔로우 or 차단 리스트를 들어가면 해당 유저의 팔로우 or 차단 리스트를 확인할 수 있습니다.
